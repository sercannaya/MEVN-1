# MEVN Stack FullStack Learning Project

## Technologies used in this project
* Front-End
  * Vue.js 2X
  * Vuetify
  * Nuxt.js
  * Cors
* Back-End
  * Node.js
  * Express.js
  * Mongoose
  * Cors
  * Aixos
* Database
  * MongoDB     

## Project Setup
`Serverside code`

```
cd apis
yarn 
yarn start
```
`Clientside code`
```
cd app
yarn 
yarn start
```
